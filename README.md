### 1.1.1 目录介绍
**bin**:        二进制文件存放目录 <br>
**etc**:        配置文件存放目录   <br>
**include**:    编译头文件存放路径 <br>
**lib**:        编译库存放路径     <br>
**src**:        存放源码路径(其中有C基础源码和项目之分)<br>
**Makefile**:   编译Makefile脚本<br>
**README.md**:  简介<br>
