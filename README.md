### 1.1.1 目录介绍
**bin**:        二进制文件存放目录
**etc**:        配置文件存放目录
**include**:    编译头文件存放路径
**lib**:        编译库存放路径
**src**:        存放源码路径(其中有C基础源码和项目之分)
**Makefile**:   编译Makefile脚本
**README.md**:  简介
